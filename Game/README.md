Factoy Escape

In this game you are a robot that should escape from the factory he's been recluded,
but beware of acid waste and other dangers.



Good luck and have fun!


Controls:

 	- A: Move left
	- D: Move right
	- SPACE: Jump
	- LSHIFT: Slide
